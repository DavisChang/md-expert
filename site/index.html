為閱讀而生

專注做好一件事：把網頁上的 Markdown 變得好讀。

🔍

到處都能偵測

原始 .md 頁面、GitHub／GitLab raw、網頁內嵌區塊、線上編輯器、本地檔案，自動辨識。

✨

好讀的渲染

標題、表格、任務清單、程式碼、目錄導覽，並與原頁樣式完全隔離（Shadow DOM）。

🔄

換頁自動偵測

支援單頁應用（SPA）與一般換頁，內容晚載入也能自動補偵測。

🗂️

多篇分頁／側欄

一頁有多份 Markdown 時，以分頁切換；數量多時改用舒適的側邊面板。

⌨️

鍵盤快捷鍵

T 切換目錄、D 切換主題、C 複製原文、Esc 關閉。亮／暗／復古主題隨選。

🔒

隱私優先

所有內容處理皆在本機完成，不蒐集網頁內容或網址。匿名統計預設關閉、可選開啟。

🔒

你的內容，留在你的瀏覽器

Markdown Expert 不會把你瀏覽的網頁內容、Markdown 內容或網址送到任何伺服器——所有渲染都在本機完成。

唯一例外是你可主動開啟的匿名使用統計（預設關閉），且只送匿名互動事件，不含任何內容或個資。
