Open source · Privacy-first · No data collection unless you explicitly opt in to anonymous analytics

Built for focused reading

A lightweight browser extension focused on one job: making Markdown on the web easier to read.

🔍

Detects Markdown everywhere

Raw .md pages, GitHub and GitLab raw files, embedded blocks, online editors, and local files are recognized automatically.

✨

Clean rendering

Headings, tables, task lists, code blocks, and table of contents navigation render inside an isolated Shadow DOM.

🔄

Navigation aware

Works across traditional page loads, single-page apps, and content that appears after the initial render.

🗂️

Multiple documents

When a page contains more than one Markdown document, switch between tabs or use the side panel for larger sets.

⌨️

Fast controls

Toggle the table of contents, switch themes, copy source Markdown, and close the reader with keyboard shortcuts.

🔒

Privacy-first

Content is processed locally in your browser. Anonymous analytics are optional and off by default.

🔒

Your content stays in your browser

Markdown Expert does not send page content, Markdown content, or URLs to any server. Rendering happens locally.

The only optional exception is anonymous usage analytics, which are off by default and never include page content or personal data.
