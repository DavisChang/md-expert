# 隱私權說明 — Markdown Expert

最後更新：2026-06-15

## 一句話
Markdown Expert 預設**不蒐集、不儲存、不傳輸**任何個人或瀏覽資料；所有內容處理都在你的瀏覽器本機完成。**唯一例外**是你主動開啟的「匿名使用統計」（預設關閉，見下）。

## 我們做什麼
- 在你造訪的網頁中，於**本機**偵測並渲染 Markdown 內容。
- 把你的偏好設定（主題、每網域展開規則、字級等）儲存在 Chrome 的 `storage.sync`，僅用於同步你自己的設定。

## 我們不做什麼
- 不會把網頁內容、Markdown 內容、網址或任何個資送到外部伺服器。
- 不使用第三方廣告 SDK。

## 選用：匿名使用統計（opt-in，預設關閉）
你可以在擴充設定頁主動開啟「分享匿名使用統計」。**只有開啟後**，擴充才會透過 Google Analytics 4（Measurement Protocol）傳送少量**匿名互動事件**，用於改善產品：

- 事件範例：偵測到 Markdown（含來源類型如 raw-page/line-editor 與篇數）、開啟閱讀視圖／側欄、工具列操作、設定變更、安裝、按「讚」。
- **絕不**傳送：網頁內容、Markdown 內容、網址、標題或任何可識別個人的資訊。
- 識別碼為隨機產生、僅存於本機的匿名 `client_id`，不關聯任何帳號。
- 你可隨時在設定頁關閉，立即停止傳送。

## 權限說明
| 權限 | 用途 |
| ---- | ---- |
| `storage` | 儲存你的偏好設定（本機 / 你的 Chrome 帳號同步）。 |
| `tabs` | 偵測分頁載入/換頁，以更新圖示徽章與重新偵測。 |
| `sidePanel` | 多篇 Markdown 時以側邊面板呈現。 |
| content script（`<all_urls>`） | 在頁面中就地偵測與渲染 Markdown；僅本機執行，不外連。 |

## 本地檔案
若你開啟對 `file://` 的存取（需於擴充頁手動開啟），擴充才能渲染本機 `.md` 檔；內容同樣僅在本機處理。
